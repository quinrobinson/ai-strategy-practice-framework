System Architecture

AI Strategy Practice Framework

Need-driven agent activation. Multi-agent sessions. Claude is the decision tree.

Entry Points
Before
Should we use AI here, and how?
Project start — problem qualification, method selection, risk mapping
During
Are we using AI in the right ways?
Mid-project — recalibration, drift detection, emerging risk flags
After
Did our AI approach actually work?
Project close — outcome vs. intention, retrospective, lessons forward
↓
Orchestrator
Claude AI — The Decision Tree
Reads the situation. Activates the right agents. Synthesizes outputs into a single recommendation.
↓
Predictor
Always active alongside every session — pattern-matches inputs against the failure mode library and surfaces risks the team hasn't hit yet.
